---
title: "Talk-me"
description: "Настройте интеграцию с Talk-me для подключения ответов с использованием ИИ"
---

<Steps>
    <Step title="Получите API ключ">
       На [портале](https://app.wikibot.pro) перейдите в раздел **Подключения** своего бота, включите интеграцию Talkme и скопируйте API ключ.
    </Step>
     <Step title="Активируйте бота в Verbox">
        Для завершения интеграции перейдите на сайт [Talk-me](https://talk-me.ru/) и завершите настройку бота, используя скопированный API ключ.
     </Step>
</Steps>
